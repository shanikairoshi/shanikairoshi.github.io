DeepUnfolding-Agent API

DeepUnfolding-Agent is the main software artefact of the PhD: a strategy-agnostic API platform for deep-unfolded adaptive control in classical, quantum, and hybrid workflows.

View GitHub repositorySee repository map

Software purpose

Adaptive controllers for classical, quantum, and hybrid optimisation workflows are often implemented as task-specific code coupled to one dataset or training loop. DeepUnfolding-Agent exposes these controllers through a stable API so external users can submit diagnostics, receive bounded control suggestions, and update controllers using feedback.

Supported capabilities

Classical control

Adaptive FL controls such as learning rate, proximal regularisation, and local training behaviour.

Quantum control

Quantum learning rate, shots, circuit depth, entanglement depth, and measurement reuse.

Hybrid workflows

Reusable controller design for classical-quantum and external optimisation loops.

External-user API

FastAPI endpoints for creation, suggestion, feedback update, experiment execution, and result retrieval.

Qiskit Aer

Simulation-based quantum experiments for fixed-versus-controlled comparisons.

IBM QPU

Optional real quantum backend workflow for selected benchmark experiments.

Core endpoints

EndpointPurpose
GET /healthCheck API service status
GET /strategiesList available controller strategies

POST /controllers/createCreate an adaptive controller
POST /controllers/{controller_id}/suggest-controlsReturn bounded control suggestions
POST /controllers/{controller_id}/update-controllerUpdate controller using feedback
POST /quantum/run-controlled-roundExecute a controlled quantum round
POST /experiments/qiskit-aer-fixed-vs-controlledRun fixed-versus-controlled Qiskit Aer experiments
