# Deployment steps for GitHub Pages

## Option A: exact desired URL

Desired URL:

```text
hhttps://shanikairoshi.github.io/
```

Required GitHub owner name:

```text
ShanikaINanayakkara
```

Required repository name:

```text
ShanikaINanayakkara.github.io
```

## Option B: keep current GitHub username

If your current username is `shanikairoshi`, create:

```text
shanikairoshi.github.io
```

Your website URL will be:

```text
https://shanikairoshi.github.io/
```

The website can still display:

```text
Shanika I. Nanayakkara
```

as the public academic identity.

## Option C: use a custom domain later

You can later use a domain such as:

```text
shanikainanayakkara.com
```

and connect it to GitHub Pages.
